What is RendaJS

Renda is a lightweight single page web & mobile application development framework. With Renda, developing web apps is an easier, faster and efficient task as we do not restrict developers to specific tools and methods. Renda follows the MVC pattern of structuring code which is a tested and trusted industry standard way of modeling projects. We employ developers using this library to follow this, as you can be assured of best practices when using MVC in your Renda project. RendaJS can be used in conjunction with any other javascript framework. You can decide to outsource some part of your project and still have RendaJS on the side for those functions you prefer RendaJS for.
