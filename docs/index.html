What is RendaJS

Renda is a lightweight single page web & mobile application development framework written in Typescript.
With Renda, developing web apps is an easier, faster and efficient task as we do not restrict developers to specific tools and methods.
Renda encorages the MVC pattern of structuring code which is a tested and trusted industry standard way of modeling projects as following a structured pattern of development is one sure way of ensuring best practices.
RendaJS can be used in conjunction with any other javascript framework. You can decide to outsource some part of your project and still have RendaJS on the side for those functions you prefer RendaJS for.

Features

  • Fast: Build Lightweight Web Applications Fast
  • Mobile Ready: Mobile App Development Using Apache Cordova or Intel SDK
  • Industrial Standard: Develop Single Page Applications With Ease
  • Flexible: Use Renda with other libraries (JQuery, VueJS, CommonJS, Bind.JS)
  • Scalable: Encorages MVC method of application development
